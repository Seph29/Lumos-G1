Key Features

What makes Lumos-G1 unique

No Bootloader Unlock

Works without unlocking bootloader, preserving device warranty.

Automated Process

Two-step system patching with reboot between phases.

Multilingual

Full support for French, English and German.

Integrity Checks

All critical files are verified with digital signatures.

Partial Restore

Restore only the system partition while keeping boot patched.

Battery Protection

Prevents patching if battery is below 20% or not charging.
